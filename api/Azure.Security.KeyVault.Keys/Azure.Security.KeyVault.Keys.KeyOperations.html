    A key stored in Key Vault may be used to protect another key, typically a symmetric content encryption key (CEK). A key stored in Key Vault may be used to decrypt a single block of data.     A key stored in Key Vault may be used to protect another key, typically a symmetric content encryption key (CEK). When the key in Key Vault is asymmetric, key encryption is used.
