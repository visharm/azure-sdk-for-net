    Interface IPartitionProcessor

    Processes events received from the Azure Event Hubs service. A single instance of a class implementing this interface will be created for every partition the associated EventProcessor owns.

    Namespace: Azure.Messaging.EventHubs.Processor
    Assembly: Azure.Messaging.EventHubs.dll
    Syntax
    public interface IPartitionProcessor
    Remarks

    Every aforementioned instance is created by a factory provided by the user in the EventProcessor constructor.
