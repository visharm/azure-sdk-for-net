    Class EventProcessor

    Constantly receives EventData from every partition in the context of a given consumer group. The received data is sent to an IPartitionProcessor to be processed.

    Remarks

    Ownership of the eventHubClient is assumed to be responsibility of the caller; this processor will delegate operations to it, but will not perform any clean-up tasks, such as closing or disposing of the instance.
